AIM To perform Categorical Data Analysis using Python, including frequency distribution, percentage analysis, cross-tabulation, grouping, filtering, and sorting of categorical variables.

THEORY Categorical Data Analysis deals with data that can be divided into groups or categories such as Grade, Gender, Department, Category, Payment Method, etc. In this experiment, the Pandas library is used for handling and analyzing such data efficiently.

Data Loading pd.read_csv() Used to load data from a CSV file into a DataFrame.

Frequency Count value_counts() Counts occurrences of each unique category. Used for Grade, Gender, Department, Category, and Payment Method.

Percentage Distribution value_counts(normalize=True) Returns proportions instead of counts.

Cross Tabulation pd.crosstab() Creates a contingency table showing relationship between two categorical variables.

Grouping Data groupby() Groups data based on one or more columns.

Creating DataFrame pd.DataFrame() Used to create a structured dataset from a dictionary.

Unique Values Analysis unique() Returns all unique values in a column. nunique() Returns the count of unique values.

Data Filtering Used to extract specific rows based on conditions.

Sorting Data sort_values() Sorts DataFrame based on a column.

Display Function display() Used to present DataFrame outputs clearly (mainly in notebooks).

Advanced Grouping Combining grouping and counting

CONCLUSION In this experiment, categorical data analysis was successfully performed using Python. Various operations such as frequency counting, percentage distribution, cross-tabulation, grouping, filtering, and sorting were implemented using Pandas functions. These techniques help in understanding patterns, relationships, and distributions within categorical datasets. The experiment demonstrates how Python can be effectively used for data analysis and decision-making based on categorical variables.
